Creating a Theme

Abstract

Staff users can store different page styles and branding configurations in the <b>Themes</b> tab. Themes control how various objects in the system appear to the user by allowing staff users to play with the <b>Color Scheme</b>, <b>Logo</b>, <b>Custom CSS</b>, <b>Header</b>, <b>Footer</b>, <b>HTML Head</b>, and <b>Supernav</b>. Additionally, <b>Advanced Settings</b> provide options for adding custom stylesheets.
Note
All URLs must originate from a secure site.
