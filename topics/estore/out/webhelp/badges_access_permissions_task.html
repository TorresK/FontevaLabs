Access Permissions

Admins can configure access permissions directly on the Item record to restrict visibility in the Community Portal. Badge assignments are granted to portal users in order to provide them with exclusive opportunities and full functionality of the Community Portal.
Note: Badges cannot be assigned to guest users nor can guest users view any item with a badge assignment. Further, if access permissions are enabled, but there are no assigned badges, access to the item is restricted to staff users and the System Administrator.
  1. Navigate to the Item record in the eStore application that requires restricted access and select the Enable Access Permissions checkbox. Click Save.
    Alternatively, navigate to the Item Class record to apply access permissions to an entire group of items.
  2. Click the Manage Permissions button on the Access Permissions related list.
  3. Select the permissible badge types and click > to grant access. Click Save.
    All available badge types are present.
Newly enabled badge types will display in the Access Permissions related list on the Item record.
