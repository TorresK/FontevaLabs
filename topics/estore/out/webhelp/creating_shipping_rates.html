Creating a Shipping Rate

Shipping Rates display in the checkout screen when a customer selects an item that requires shipping. The Shipping Rate is calculated based upon:
  • Shipping Carrier
  • Delivery Method
  • Shipping Region
  • Minimum and Maximum Item Weights or Item Value

    Users can set up multiple Shipping Rates or define a single Flat Rate per item or a Flat Rate per order.

  • Create the Shipping Carrier and Delivery Method
  • Create the Shipping Region
  1. Navigate to the Items tab in the eStore app and click New.
  2. Enter the Name for the Shipping Rate and enter a Display Name.
    The Display Name appears on the eStore checkout screen and on the customer's receipt.
  3. Identify the Shipping Carrier using the Item Class lookup.
  4. Enter the cost of shipping applied to the item in the Item Price field.
  5. Select the Business Group and check the Is Active and Is Shipping Rate checkboxes.
  6. Select the Shipping Method associated with the new Shipping Rate using the lookup.
  7. Select the Shipping Region.
  8. Enter the shipping item's Minimum Weight and Maximum Weight.
    Alternatively, if the Shipping Rate Basis set on the Business Group record is Item Value, then enter the Minimum Value and Maximum Value.
  9. Select the Item Weight UOM (Unit of Measurement) using the picklist.
    The Item Weight UOM value is used when creating an eStore item that requires shipping and when creating a Shipping Rate item. Admins do not have to use the same units for items and Shipping Rates. For example, the Shipping Rate might be defined in pounds and the Item Weight in ounces. The system performs the conversion on the backend and will apply the correct rate in the eStore checkout, regardless of the unit.
  10. The new Shipping Rate inherits the GL Accounts of the associated Shipping Carrier. To override and link to an alternative account, select the GL account using the Income Account and Refund/Credit Account lookups.  Click Save.
