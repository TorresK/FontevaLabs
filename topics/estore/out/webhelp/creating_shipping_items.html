Creating an Item Requiring Shipment

Once Shipping Rates are created, identify the eStore merchandise that requires shipping.
Note: If using the same Item Weight UOM across shipping items, staff users can set it as the default value in the picklist via:
  • Setup > Create > Objects > Item > Item Weight UOM > Picklist Values related list > Click Edit on the preferred unit > Check the Default checkbox > Save.
  • Create the Shipping Carrier and Delivery Method
  • Create the Shipping Region
  • Create a Shipping Rate
  1. Navigate to the Items tab in the eStore app and click New.
    Alternatively, select an existing item and click Edit.
  2. Check the Is Active and Require Shipping checkboxes.
  3. Enter the Item Weight.
    Alternatively, if the Shipping Rate Basis set on the Business Group record is Item Value, then only the Item Price is required.
  4. Select the Item Weight UOM and click Save.
    When the Shipping Rate Basis is set to Item Weight, not identifying the Item Weight UOM will result in incorrect Shipping Rates displayed to customers. Leave Item Weight UOM blank if the Shipping Rate Basis is Item Value.
Shipping is enabled on the item.
Note: If the Shipping Rate Basis is set to Item Value, staff users can still charge shipping on a $0.00 item via:
  • Checking the Require Shipping checkbox on the Item record
  • Entering "0" as the Minimum Value on the Shipping Rate Item record.
