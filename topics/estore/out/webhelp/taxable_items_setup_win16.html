Setting up a Taxable Item

System Admins must configure their items to apply tax.
  • Set up the Tax Locale
  • Set up the Tax Class
  • Set up the Tax Rate
  1. Navigate to the individual Item record by clicking the Items tab in the eStore app.
  2. Check the Is Taxable checkbox.
  3. Identify the Tax Class using the lookup and click Save.
Taxes can now be applied to items.
