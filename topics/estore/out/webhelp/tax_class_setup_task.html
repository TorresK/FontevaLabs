Setting up a Tax Class

The tax class for items is set up through the Item Class record and groups rates together according to their related tax class.
  1. Navigate to the Item Classes tab in the eStore app and click the New button.
  2. Enter the Name of the Item Class and select the Is Active checkbox.
  3. Select the Is Tax checkbox and click the Save button.
A new item is created in the Items related list called Default Tax Rate, with a value of 0.00.
Note: Each Tax Class can only have one Default Tax Rate.
