Setting up the Tax Locale

Tax locales refer to the countries, provinces, states, cities, and zip codes that may have various taxation policies related to the selling of products. System Admins must set the tax locale to account for the variations between multijurisdictional tax rates.
  • Configure Organization Default Tax Settings
  • Designate the GL Account for Tax Liabilities
  1. Navigate to the Tax Locales tab in the eStore app and click the New button.
  2. Enter the Name of the tax locale.
  3. Enter the API name of the Sales Order Line field the system is to evaluate for the tax locale in the the Tax Locale Field.
    Navigate through the Setup to the Custom Object page for the Sales Order Line to identify field API names.
  4. Enter the field values defined in the Tax Locale Field into the Locale Values (CSV) field.
  5. Designate the tax liability account using the Tax Account lookup field. Click Save.
The tax locale is available to configure the tax rate for items.
