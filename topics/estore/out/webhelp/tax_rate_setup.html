Setting up a Tax Rate

System Admins can create multiple tax rates to apply to items based on their related tax locales.
  • Set up a Tax Locale
  • Set up a Tax Class

Combined with the Sales Order Line's Sale Price field, the tax rate is used to calculate the tax amount for the related item.

  1. Navigate to the Items related list on the Tax Class record and click the New button.
  2. Select the Is Active checkbox.
  3. Link the Tax Locale lookup field to the related item and select the Is Tax checkbox.
  4. Enter a value in the Tax Percent field and click Save.
When purchasing a taxable item, tax is applied to:
  • Sales Order Line(s)
  • Invoice Line(s)
  • Receipt Line(s)
  • ePayment Line(s)
