Creating a Catalog

Catalogs allow staff and portal users to shop and create orders and browse current inventory. Staff users can add items to the backend Shopping Cart and customize catalogs by adding images and descriptions to catalog items.
Note: A catalog must be created in order for an item to display in the eStore in the Community Portal.
  • Create a Store
  • Create an Item Class and Items
  • For subscription based catalogs, make sure to Create a Subscription Item
  1. Navigate to the Catalogs related list on the associated Store record and click the New Catalog button.
  2. Enter the Catalog Name and Store.
    Select the Parent Catalog from the lookup in order to link the newly created catalog to a Parent Catalog record. Item Catalog hierarchies organize products and services available for sale and streamline the Shopping Cart experience.
  3. Select the Is Published checkbox to make the catalog available for display via the API. Click Save.
    Enter a Description for the catalog.
The catalog is created and added to the Store in the Community Portal, but will not have any items to display until they are added to the catalog.
  • Add items to the catalog:
  1. Navigate to the Items related list on the Catalog record and click New Catalog Item.
  2. Select the Item using the lookup and check the Is Featured and Is Published checkboxes. Click Save.

  • Once items are added to the catalog and a subscription is purchased, Community Portal users can Manage the Subscription.
