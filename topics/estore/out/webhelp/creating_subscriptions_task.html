Creating Subscription Items

The subscription item is the consideration granted to the subscription's beneficiaries. For example, if an organization hosts a weekly webinar for subscribers, the subscription item is the webinar.

For subscription items with multiple subscription plans, best practices recommend setting the highest price as the product's default price; the default price is displayed in the Store and on the Item Detail page.

Admins must create a price rule to link each additional, cheaper plan to the subscription item; additional prices display in the Subscription Plan modal after the item is added to the cart.

  • Create a Subscription Plan
  • Create an Item Class for Subscriptions
  1. Navigate to the Item Class tab in the eStore app and select the related Item Class record.
  2. Click the New Item button on the Items related list.
  3. Enter the Item Name and Item Price. Click Save.
    Best practices recommend entering the Display Name, Line Description, and Image URL.
    Note: If there is a definitive number of subscription beneficiaries, check the Restrict Number of Assignments checkbox and enter the value into the Maximum Number of Assignments field.
  4. Navigate to the Subscription Plans related list and click Manage Subscription Plans.
  5. Select the applicable Subscription Plan and mark it as the Default Subscription Plan. Click Save.
    Multiple Subscription Plans can be selected, but only one Subscription Plan can be marked as Default.
  6. Enable the Is Active checkbox and click Save.
The subscription item is created and assigned to one or more subscription plans. Any number of subscription items can be added to a single subscription plan.
