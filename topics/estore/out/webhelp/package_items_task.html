Creating Package Items

Package Items allow staff users to bundle single items together for sale and upsell items, single or group. Examples include:
  • T-shirt and a book with the purchase of an event ticket
  • Gift with the purchase of a membership
  • Additional item upon purchase
Note: The hierarchy of package items is one level deep. For example, an item can have several package items, but those packages items cannot have additional package items.
  • Create an Item
  1. Navigate to the Items tab in the eStore app and select an existing item.
  2. Click the New button on the Package Items related list.
  3. Use the picklist to select:
    • Single Item
    • Group of Items

Single Item

  1. Select the Item using the lookup in order for it to be included with the purchase.
    Check the checkbox to always include the selected item as part of the package.
  2. Enter the quantity per Item allowed for purchase.
  3. Enter the Display Label that is shown to registering users in the Community Portal and the Shopping Cart. Click the Save Package Item button.
    If necessary, enter the Display Instructions.

Group of Items

  1. Select the Item Class using the lookup.
    Check the checkbox to require at least one Item in the group to be purchased as part of this package.
  2. Enter the quantity per Item allowed for purchase.
  3. Enter the Display Label that is shown to registering users in the Community Portal and the Shopping Cart. Click the Save Package Item button.
    If necessary, enter the Display Instructions
The Package Item is created.
