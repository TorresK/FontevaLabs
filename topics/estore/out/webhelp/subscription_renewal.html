Renewing a Subscription

Subscriptions are renewed directly from the Sales Order record. Upgraded subscriptions or memberships cannot be renewed.
  1. Navigate to the Subscriptions tab in the eBusiness app and select the Subscription ID to be renewed.
  2. Click the Renew button.
    A new Sales Order is automatically created, along with a new Sales Order Line. The Posting Entity is set to Receipt and the Schedule Type is set to Simple Receipt.
  3. Select Closed from the Status picklist and click Save.
    If the Posting Entity is set to Receipt, payment is required in order to complete the renewal request. Click the Sales Receipt button, enter payment information, and click Pay.
    The Posting Status on the Sales Order record automatically updates to Posted to reflect payment.
The subscription is renewed and the transaction is journaled into the GL.
Note: Renewing a subscription does not create a new subscription unless the item is changed on the Sales Order Line record. This renewal is interpreted as an Upgrade, therefore a new subscription is created with a future start date.

  • Amend Subscriptions
  • Modify Subscriptions
