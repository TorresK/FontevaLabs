Creating a Flat Shipping Rate

Rather than create multiple rates to account for different Shipping Regions and item weights/values, Admins can create a simple Flat Rate per item or for the order total.
  • Create the Shipping Carrier and Delivery Method
  1. Navigate to the Items tab in the eStore app and click New.
  2. Enter the Name for the Shipping Rate.
  3. Enter a Display Name. 
    The Display Name appears on the eStore checkout screen and on the customer's receipt.
  4. Identify the Shipping Carrier using the Item Class lookup.
  5. Enter the cost of shipping applied to the item in the Item Price field.
  6. Select the Business Group and check the Is Active, Is Shipping Rate, and Is Default Shipping Rate checkboxes.
    Admins cannot specify a Shipping Region when the Is Default Shipping Rate is enabled.
  7. Select the Delivery Method associated with the new Shipping Rate using the lookup.
  8. Enter a value of "0" into the Minimum Weight field and enter a large value into the Maximum Weight field. Ensure the Maximum Weight has a value large enough to accommodate all possible shipping items.
    If the Shipping Rate Basis is set to Item Value, then enter the Minimum Value and Maximum Value.
  9. Select the Item Weight UOM (Unit of Measurement) using the picklist.
    The Item Weight UOM value is used when creating an eStore item that requires shipping and when creating a Shipping Rate item. Admins do not have to use the same units for items and Shipping Rates. For example, the Shipping Rate might be defined in pounds and the Item Weight in ounces. The system performs the conversion on the backend and will apply the correct rate in the eStore checkout, regardless of the unit.
  10. The new Shipping Rate inherits the GL Accounts of the associated Shipping Carrier. To override and link to an alternative account, select the GL account using the Income Account and Refund/Credit Account lookups.  Click Save.
