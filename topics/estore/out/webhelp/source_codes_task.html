Creating a Source Code

Source Codes can be linked to a Business Group for reporting concerns and a channel in order to track transactions. Active source codes are associated with Campaign Pages and Price Rules.
  1. Navigate to the Source Codes tab in the eStore app and click the New button.
  2. Enter the name of the Source Code and use the lookup to identify the Business Group.
    Linking the Source Code to the applicable Business Group furthers reporting goals. Optionally, select the Channel in order to track Source Code transactions.
  3. Check the Active checkbox and click Save.
