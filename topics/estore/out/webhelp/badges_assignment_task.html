Badge Assignment Automation

Badges can be automatically assigned and unassigned to both Accounts and Contacts based on assignment. When a Sales Order is posted, the system will check all related items. If any of the related items has a Badge Workflow related list, the system will create Badges automatically depending on the Badge Workflow Type.
Note: If assigned to an Account, all Contacts under that Account will automatically be assigned a Badge.
  • An active Badge Type record
  • An active Item record to assign the Badge
  1. From the Badge Type record, navigate to the Workflow related list and click the New Badge Workflow button.
  2. Enter the Badge Workflow Name.
  3. Select an item to assign the Badge to from the Item lookup.
    The Badge Type field auto-populates with the name of the Badge Type record for which the workflow is being created.
  4. Select the Type:
    • Line Contact-Assigns the Badge to the purchasing customer as defined in the Contact lookup on the Sales Order Line.
    • Assigned Contacts-Grants the Badge to all active and assigned contacts.
    • Line Account-Assigns the Badge to the purchasing account as defined in the Account lookup on the Sales Order Line.
    Optionally, select the Auto Expire checkbox to automatically unassign the Badge from the Contact record when the Assignment record becomes inactive.
  5. Create a new Sales Order for an account or contact with a Sales Order Line for the item that the Badge is to be assigned.
    Complete the purchase of the item, which will close and post the sales order.
The Badge is assigned to the specified Account or Contact record and is displayed in the Badges related list.
