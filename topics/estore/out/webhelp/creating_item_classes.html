Creating an Item Class

Item Classes group and set options for items at the class level. All items must have an Item Class.
Note: The system automatically creates Item Classes for Event ticket types, schedule items, and sponsors, if enabled on the Event record.
  1. Navigate to the Item Classes tab in the eStore app and click the New button.
  2. Enter the Name of the Item Class.
    Identify the Business Group via the lookup and enter the Description for the Item Class.
  3. Identify any relevant Item Class Attributes.
  4. Select the Is Active checkbox and click Save.
