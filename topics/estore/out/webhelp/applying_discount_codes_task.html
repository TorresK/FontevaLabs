Applying a Discount Code

Discount Codes give organizations the functionality to provide customers reduced prices on specified items or events that are added to their Shopping Cart. System Admins and staff users can apply a Discount Code to an item on the the Sales Order record by populating the Source Code field.
  1. Create a Source Code
  2. Create a Price Rule
  3. System Admins must enable Discount Codes in their environment:
    • Navigate to the username drop down list and click Developer Console.

    • Select the Debug topbar menu item and click the Open Execute Anonymous Window from the drop down list.

    • Enter the following script into the Enter Apex Code window: Framework.AppUpdate app = Framework.AppUpdate.get('DiscountCode'); app.isActivated = true; Framework.Config.push(app);

      Click Execute and close the Force.com Developer Console window.

User Experience

  1. Authenticated users will log in to the Community Portal and click the Store tab in the navbar.
  2. Select the item associated with the Discount Code and click Add to Cart.
    In the Shopping Cart modal, Community Portal users can click Update Cart or proceed to Checkout. Click Checkout.
  3. Both authenticated and unauthenticated users will enter the Discount Code into the Order Total pane on the Checkout page and click Apply.
    Only one Discount Code can be applied per transaction.
    The Checkout page is updated to reflect the applied discount.
  4. Complete Payment Information and click Pay.
Discount Codes are successfully applied allowing backend users and both authenticated and unauthenticated users the benefit of a reduced price on an item or event.
Note: In order to pass a Discount Code on an event to others, navigate to the Source Code record and copy the Source Code ID from the URL. Next, append the Source Code ID to the Community URL: {!communityurl}/Eventapi__Router?event={!event.id}&site={!site.id}&sourcecode={!Source_Code__c.id}
