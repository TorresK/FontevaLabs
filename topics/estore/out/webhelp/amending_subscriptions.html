Amending Subscriptions

System Admins will amend subscriptions for several reasons, such as increasing the seats for a membership and purchasing additional subscriptions. For example, if a customer purchases a Termed Subscription Plan lasting six months for 10 seats, and realizes after two months that three additional seats are needed, the subscription requires an amendment. In this case, the total cost of the three extra seats can be prorated to reflect the four months the subscription is active. Although proration is exclusive to Calendar subscription plans, System Admins will check the Enable Proration checkbox and select the Proration Rule when making a necessary amendment to a Termed subscription that allows for the distribution of the purchase price over a set period of time.
  1. Navigate to the Subscriptions tab in the eBusiness app and click the Subscription ID to return the Subscription record.
  2. Click the Amend Subscription button.
    A new Sales Order is created.
  3. Navigate to the Sales Order Lines related list and copy the Sales Order Line ID.
    The copied Sales Order ID is the number of the Parent Sales Order Line.
  4. Click the New Sales Order Line button.
  5. Select an Item that is different from the item on the Parent Sales Order via the lookup and check the Is Amendment checkbox.
    Instead of selecting the same item, create an independent Sales Order.
  6. Paste the copied ID into the Parent Sales Order Line field and click Save.
  7. Click the Sales Order ID to return the Sales Order record.
  8. Update the Status to Closed and click Save.
    1. If the Posting Entity is set to Receipt, payment is required in order to complete the renewal request. Click the Sales Receipt button, enter payment information, and click Pay. The Posting Status on the Sales Order record automatically updates to Posted to reflect payment.
The subscription is amended.
Admins can additionally:
  • Modify Subscriptions
  • Renew Subscriptions
